# Maxim Koshel Portfolio

Static GitHub Pages portfolio for Maxim Koshel, Senior DevOps Engineer.

The site is intentionally lightweight: plain HTML and CSS, no build step, no backend, and no runtime dependencies. It presents public-safe experience, impact, case-study summaries, and contact links for DevOps, SRE, platform engineering, and infrastructure roles.

Published with GitHub Pages:

https://maximkoshel.github.io/portfolio/

## Stack

- HTML
- CSS
- GitHub Pages

## Privacy note

The portfolio content avoids raw ticket IDs, internal service names, incident details, private URLs, and environment-level implementation details. Personal documents such as grades, discharge certificates, and old resume exports are intentionally not published from the current site tree.
